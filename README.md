Project Astra is a real-time monitoring system designed to address critical urban challenges related to air pollution and road deterioration. The system uses low-cost sensors to collect environmental and road-condition data, transmits it through a mobile gateway using Bluetooth, and processes the data on the cloud using machine learning models. By leveraging standard network services such as DNS and SMTP, Project Astra ensures efficient data routing, real-time analysis, and timely alert delivery for smarter urban decision-making.
